Sound Ideas

This is currently a holding page while we pull together our content from various locations on GitHub. If you're looking for the test page with the QR codes and instrument examples, hold tight! They'll be back up here soon.

The website a set of open source instruments developed by Blurring the Boundaries Arts in collaboration with the National accessArts Centre.

We have developed these instruments through a series of workshops, and we are now making the tools used to create them available to the public through a set of JavaScript libraries. We hope that you will find them useful, and that you will share your creations with us.
